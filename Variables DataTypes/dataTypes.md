# DATA TYPES

- JavaScript has the following basic data types:
    - Primitive Types:

        1. String
        2. Number
        3. Bigint
        4. Boolean
        5. Undefined
        6. Null
        7. Symbol
        8. Object

    - Reference Types:

        1. An object
        2. An array
        3. A date


### Example

```js
// Undefined:
let x;
let y;

// Numbers:
let weight = 50;
let height = 1.55;

// Strings:
let firstName = "Cường";
let middleName = "Anh";
let lastName = "Nguyễn";

// Booleans
let male = true;
let female = false;

// Object:
const person = {
    iD: "nac445",
    firstName: "Cường",
    lastName: "Nguyễn",
};

// Array object:
const vegetable = ["Tomato", "Onion", "Potato"];

// Date object:
const dateOfBirth = new Date("2005-04-04");
```

- JavaScript Types are Dynamic
